I have been experimenting with Virtual Desktops in cachy os,

in combination with a tiling manager

with some useful shortcuts to go between tiles and desktops, 

its amazing...

gets even better if you have an external monitor

For now i like having 3 desktops:

## productivity, logistics

- noteshub
- btop to see system usage

## project desktop

- whatever you are working right now

## communications - leisure

- stuff like whatsapp, discord, slack, gmail
- also youtube or youtube music.


## browser workspaces

this also works as a classification of workspaces in browsers like vivaldi
